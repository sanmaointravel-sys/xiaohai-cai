# 🍳 小孩菜

> 帮家长快速判断一道菜适不适合孩子：过敏原、营养成分、疾病禁忌一目了然

## 功能概览

### 核心功能
- 👶 **孩子档案管理**：基本信息、过敏原、不爱吃的、当前身体状态
- 📖 **菜谱成分分解**：每道菜拆解成原料 + 成分分析
- 🚦 **智能筛选**：一键判断某道菜适不适合当前孩子
- 🩺 **生病饮食指南**：感冒/发烧/腹泻等常见病的饮食建议
- 📋 **饮食记录**：记录孩子每餐吃了什么

## 技术架构

```
微信小程序(用户端)
        ↓
   Flask 后端API
        ↓
   SQLite 数据库
   (食材库 + 菜谱库 + 孩子档案)
```

## 数据库结构

- `children` - 孩子档案
- `allergy_types` - 过敏原配置
- `child_allergies` - 孩子过敏原关联
- `child_dislikes` - 孩子不爱吃的食材
- `illnesses` - 常见病及禁忌
- `child_health_status` - 孩子当前疾病状态
- `ingredients` - 食材成分数据（约80种）
- `recipes` - 菜谱（20道）
- `recipe_ingredients` - 菜谱原料关联
- `meal_logs` - 每日饮食记录

## 启动方式

### 后端
```bash
cd backend
pip install flask flask-cors
python app.py
```
后端地址：http://localhost:5000

### 小程序
微信开发者工具导入 `client/` 目录，填入后端地址即可预览

## 页面说明

| 页面 | 功能 |
|------|------|
| 首页 | 智能筛选：能吃/慎选/不能吃 |
| 菜谱 | 浏览全部菜谱、搜索 |
| 孩子 | 管理孩子档案、过敏原、疾病 |
| 记录 | 记录每日饮食 |
