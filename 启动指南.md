# 小孩菜 - 启动指南

## 一、先拉取最新代码

打开 **CMD**，输入：

```bash
cd C:\Users\Administrator\Desktop
git clone https://github.com/sanmaointravel-sys/xiaohai-cai.git
```

> 如果已经有仓库了，输入：
> ```bash
> cd C:\Users\Administrator\Desktop\小孩菜
> git pull origin main
> ```

---

## 二、启动后端（双击运行）

桌面上找到 **`启动后端.ps1`**，右键 → **"用 PowerShell 运行"**

✅ 成功会看到：
```
========================================
  小孩菜 - Backend Launcher
========================================

Python: C:\Program Files\Python311\python.exe
Python 3.11.x

[1/3] Installing flask & flask-cors...
Done
[2/3] Starting server...
URL:    http://localhost:5000
```

如果提示"无法加载文件"，管理员打开 PowerShell，输入：
```bash
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```
然后重新双击运行 `启动后端.ps1`

---

## 三、修改小程序 API 地址

微信开发者工具导入 `client/` 目录后，打开 **`app.js`**，把第三行改成你的电脑IP：

```javascript
// 打开 CMD，输入 ipconfig，找到"IPv4 地址"
const API_BASE = 'http://你的电脑IP:5000/api';
```

例如：`const API_BASE = 'http://192.168.1.100:5000/api';`

---

## 四、打开微信开发者工具

导入路径：`C:\Users\Administrator\Desktop\小孩菜\client\`

---

## 五、常见问题

**Q: PowerShell 闪退？**
→ 打开 PowerShell（Win+X → A），手动运行：
```bash
cd C:\Users\Administrator\Desktop\小孩菜
.\启动后端.ps1
```

**Q: 后端提示"端口被占用"？**
→ 端口5000被其他程序占用，关闭后重新运行即可

**Q: 小程序显示"后端未启动"？**
→ 检查 app.js 里的 IP 地址是否正确，IP 地址可通过 CMD 输入 `ipconfig` 查看
